Discordのコマンド一覧

右側に表示例

1.スポイラー(ネタバレ防止)

|| spoiler || このように文字を"||"で囲むとスポイラーになり、クリックすると表示されるようになります

押してみて！→
spoiler

さらに文字の前に、/spoiler と入力してもスポイラーになります

2.見出し

# h1 このように文字の前に"#"を入力すると、大きくなります

h1

## h2 ### h3 大きさは4段階あり、"#"を多くすればするほど小さくなります。-# -h で一番小さくなります

h2

h3

-h

3.斜体

_(oblique)_ このように文字を" _ "もしくは" * "で囲むと、斜体になります

oblique

さらに文字を入力する前に、command+i/ctrl+i を入力することでテンプレートが出てきます。(mac/windows限定)

4.太字

** bold ** このように文字を" ** "で囲むと、太字になります

bold

5.打ち消し線

~~strikethrough~~ このように文字を" ~~ "で囲むと、打ち消し線になります

strikethrough

6.下線

__underline__ このように文字を" __ "で囲むと、下線になります

underline

さらに文字を入力する前に、command+u/ctrl+u を入力することでテンプレートが出てきます。

7.リスト

* list このように文字の先頭に" * "もしくは" - "を入力すると、リストになります

  • list1
  • list2
  • list3

8.コードブロック

```
code
``` このように文字を" ``` "で囲むと、インラインコードブロックになります

code
io::stdin()
.read_line(&mut input)
.expect("Failed to read line");

また、コードブロックの中だとshift+enterではなく、enterを押すと改行されます

8.5.シンタックスハイライト

コードブロックを入力する際、最初の```の後に、任意の言語を入力することで、シンタックスハイライトになります

```javascript
code
for ( let i = 10; i >= 0 ; i-- )
```

code
for ( let i = 10; i >= 0 ; i-- )

9.インラインコードブロック

`code` このように文字の前に" ` "を入力すると、インラインコードになります

code

10.マスクされたリンク

[リンクのテキスト](URL)このように入力すると、マスクされたリンクになります。（ハイパーリンクにはなりません）

[ニコニコ動画](https://youtube.com/)

ニコニコ動画

注意：このリンクをクリックした際、警告が表示されるようになっています

11.引用

> 引用 このように" > "もしくは" >>> "を先頭に入力すると、引用になります

引用

12.絵文字

:smile: このように絵文字を入力すると、絵文字になります

😄

さらに文字を入力する前に、command+e/ctrl+e を入力することで絵文字一覧が見えます（mac/windows限定）

小ネタ：\をコマンドの前に入力すると、コマンドを無視して送信することができます
